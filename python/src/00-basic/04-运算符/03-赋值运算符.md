# 赋值运算符

假设变量 a 为 10，变量 b 为 20：

运算符 | 描述 | 实例
-- | -- | --
= | 简单的赋值运算符 | c = a + b 将 a + b 的运算结果赋值为 c
+= | 加法赋值运算符 | c += a 等效于 c = c + a
-= | 减法赋值运算符 | c -= a 等效于 c = c - a
\*= | 乘法赋值运算符 | c \*= a 等效于 c = c * a
/= | 除法赋值运算符 | c /= a 等效于 c = c / a
%= | 取模赋值运算符 | c %= a 等效于 c = c % a
\**= | 幂赋值运算符 | c **= a 等效于 c = c ** a
//= | 取整除赋值运算符 | c //= a 等效于 c = c // a

```python
a = 21
b = 10
c = 0

c = a + b
print ("1 - c 的值为：", c)

c += a
print ("2 - c 的值为：", c)

c *= a
print ("3 - c 的值为：", c)

c /= a 
print ("4 - c 的值为：", c)

c = 2
c %= a
print ("5 - c 的值为：", c)

c **= a
print ("6 - c 的值为：", c)

c //= a
print ("7 - c 的值为：", c)
```
